## No tests found for given includes:

> 현재 클래스에 테스트를 발견할 수 없음.

- @Test 어노테이션을 잡지 못하는 상태
- Settings > Build,Execution,Deployment > Build Tools > Gradle > "Run tests using: IntelliJ IDEA" 로 변경
