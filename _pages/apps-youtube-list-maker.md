YouTube List Maker는 사용자가 입력한 가수 이름 또는 검색어를 기반으로 YouTube 및 YouTube Music에서 열 수 있는 음악 재생 링크를 만들어 주는 앱입니다.

본 앱은 앱 자체 계정을 만들지 않으며, Google 로그인 또는 YouTube 계정 로그인을 요구하지 않습니다. 또한 사용자의 YouTube 계정에 재생목록을 직접 생성하거나 저장하지 않습니다.

## 주요 기능

- Google 로그인 없음
- YouTube 계정 권한 요청 없음
- 저장형 YouTube 재생목록 생성 없음
- YouTube / YouTube Music 임시 재생목록 링크 생성
- 기본 10곡
- 최대 30곡
- Shorts 제외
- 120초 미만 영상 제외
- 중복 곡 제거
- 공식 음원 / Topic / 공식 MV 우선
- 서버 메모리 캐시
- Known Artist 사전 등록
- Cloud Run 저비용 배포

## 사용 방법

1. 앱에서 가수 이름 또는 검색어를 입력합니다.
2. 생성할 곡 개수를 입력합니다.
3. 재생 링크 생성 버튼을 누릅니다.
4. 생성된 곡 목록을 확인합니다.
5. YouTube 또는 YouTube Music에서 엽니다.
