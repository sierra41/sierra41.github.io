시행일: 2026년 6월 12일

YouTube List Maker는 사용자가 입력한 검색어를 기반으로 사용자의 YouTube 계정에 재생목록을 생성하는 앱입니다. 본 개인정보처리방침은 YouTube List Maker가 어떤 정보를 사용하고, 어떻게 처리하는지 설명합니다.

YouTube List Maker는 앱 자체 계정을 생성하지 않으며, Google 계정 로그인은 YouTube 재생목록 생성 기능 제공을 위한 인증 수단으로만 사용합니다.

## 1. 수집 및 사용하는 정보

YouTube List Maker는 앱 기능 제공을 위해 다음 정보를 사용할 수 있습니다.

- Google 계정 인증 정보
- YouTube API 접근 권한
- 사용자가 입력한 가수 이름 또는 검색어
- 생성된 YouTube 재생목록 및 영상 정보

YouTube List Maker는 사용자의 Google 비밀번호를 수집하거나 저장하지 않습니다.

## 2. 정보 사용 목적

수집 또는 접근한 정보는 다음 목적에만 사용됩니다.

- Google 계정 로그인
- 사용자의 YouTube 계정에 재생목록 생성
- YouTube 영상 검색
- 생성된 재생목록을 YouTube 또는 YouTube Music에서 열기

## 3. 정보 저장 여부

YouTube List Maker는 사용자의 Google 계정 정보, access token, refresh token, YouTube 재생목록 데이터, 검색 기록을 서버에 영구 저장하지 않습니다.

앱 서버는 사용자의 요청을 처리하기 위해 일시적으로 access token과 검색어를 사용하며, 요청 처리가 끝난 뒤 별도로 보관하지 않습니다.

## 4. 제3자 서비스 사용

YouTube List Maker는 기능 제공을 위해 Google API 및 YouTube Data API를 사용합니다.

사용자는 Google 계정 로그인 및 권한 동의 과정에서 Google의 개인정보처리방침과 YouTube API 서비스 약관의 적용을 받을 수 있습니다.

- Google 개인정보처리방침: <https://policies.google.com/privacy>
- YouTube API 서비스 약관: <https://developers.google.com/youtube/terms/api-services-terms-of-service>

## 5. 권한 철회 및 데이터 삭제

사용자는 Google 계정 권한 관리 페이지에서 YouTube List Maker에 부여한 접근 권한을 철회할 수 있습니다.

Google 계정 권한 관리 페이지:
<https://myaccount.google.com/permissions>

YouTube List Maker는 사용자의 데이터를 서버에 영구 저장하지 않습니다.

이미 생성된 YouTube 재생목록은 사용자의 YouTube 계정에 저장되며, 사용자가 YouTube 또는 YouTube Music에서 직접 삭제할 수 있습니다.

## 6. 아동 개인정보

YouTube List Maker는 만 13세 미만 아동을 대상으로 하지 않습니다. 앱은 아동의 개인정보를 고의로 수집하지 않습니다.

## 7. 보안

YouTube List Maker는 사용자의 정보를 보호하기 위해 HTTPS 통신을 사용합니다. 앱 서버는 요청 처리에 필요한 범위 내에서만 Google access token을 사용합니다.

## 8. 개인정보처리방침 변경

본 개인정보처리방침은 앱 기능, 법적 요구사항, 서비스 운영 방식 변경에 따라 수정될 수 있습니다. 변경 시 본 페이지의 시행일을 갱신합니다.
