시행일: 2026년 6월 14일

YouTube List Maker는 사용자가 입력한 가수 이름 또는 검색어를 기반으로 YouTube 및 YouTube Music에서 열 수 있는 음악 재생 링크를 만들어 주는 앱입니다.

본 앱은 앱 자체 계정을 만들지 않으며, Google 로그인 또는 YouTube 계정 로그인을 요구하지 않습니다. 또한 사용자의 YouTube 계정에 재생목록을 직접 생성하거나 저장하지 않습니다.

## 1. 수집 및 사용하는 정보

본 앱은 기능 제공을 위해 다음 정보를 사용할 수 있습니다.

- 사용자가 입력한 가수 이름 또는 검색어
- 검색 결과로 생성된 YouTube 영상 ID, 제목, 채널명, 썸네일 정보
- 생성된 YouTube 및 YouTube Music 재생 링크

본 앱은 사용자의 이름, 이메일 주소, Google 계정 정보, 비밀번호, 연락처, 위치 정보, 결제 정보 등을 수집하지 않습니다.

## 2. 정보 사용 목적

입력된 정보는 다음 목적에만 사용됩니다.

- 음악 영상 또는 음원 후보 검색
- YouTube 및 YouTube Music에서 열 수 있는 재생 링크 생성
- 중복 결과 제거 및 검색 결과 품질 개선
- 앱 기능 제공 및 오류 처리

## 3. 정보 저장 여부

본 앱은 사용자의 개인정보를 서버에 영구 저장하지 않습니다.

서버는 비용과 YouTube API 사용량을 줄이기 위해 검색어와 검색 결과를 일시적으로 메모리 캐시에 저장할 수 있습니다. 이 캐시는 개인을 식별하기 위한 목적이 아니며, 서버 재시작 또는 일정 시간이 지나면 삭제됩니다.

캐시 보관 기준은 다음과 같습니다.

- 인기 검색어 결과: 최대 7일
- 일반 검색어 결과: 최대 1일
- 결과 없음 또는 실패 결과: 최대 1시간

## 4. 제3자 서비스 사용

본 앱은 음악 검색 및 링크 생성을 위해 YouTube Data API를 사용할 수 있습니다.

사용자가 생성된 링크를 열면 YouTube 또는 YouTube Music 앱/웹사이트로 이동할 수 있으며, 이 경우 Google 및 YouTube의 개인정보처리방침과 서비스 약관이 적용될 수 있습니다.

- Google 개인정보처리방침: <https://policies.google.com/privacy>
- YouTube 서비스 약관: <https://www.youtube.com/t/terms>
- YouTube API 서비스 약관: <https://developers.google.com/youtube/terms/api-services-terms-of-service>

## 5. 데이터 삭제

본 앱은 사용자의 계정이나 개인정보를 저장하지 않으므로 별도의 계정 삭제 절차가 필요하지 않습니다.

사용자가 입력한 검색어와 검색 결과 캐시는 일정 시간이 지나면 자동으로 삭제되며, 서버 재시작 시에도 삭제될 수 있습니다.
